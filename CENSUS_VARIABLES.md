## NOTES:
- All variables use the format: `https://api.census.gov/data/2022/acs/acs5?get=VARIABLE_CODE&for=zip%20code%20tabulation%20area:ZIPCODE&key=YOUR_KEY`
- Some statistics require calculations (percentages, rates) using multiple variables
- Not all variables may be available for all ZIP codes
- Data is from 2022 ACS 5-Year Estimates (most comprehensive)

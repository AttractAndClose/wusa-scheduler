# Quick deployment guide

## GitHub Repository
✅ Created: https://github.com/AttractAndClose/wusa-scheduler
✅ Code pushed successfully

## Deploy to Vercel

### Option 1: Via Dashboard (Easiest)
1. Go to: https://vercel.com/new
2. Sign in with GitHub
3. Import: AttractAndClose/wusa-scheduler
4. Click "Deploy" (settings are auto-detected)

### Option 2: Via CLI
```bash
vercel login
vercel --prod
```

## After Deployment
Your live URL will be: `https://wusa-scheduler-*.vercel.app`

You can also add a custom domain in Vercel settings.
